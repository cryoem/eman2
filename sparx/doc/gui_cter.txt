= Name =
sxgui_cter - CTF Assessment: GUI tool to assess and sort micrographs based on their estimated CTF by sxcter.


= Usage =


''usage in command line''

sxgui_cter.py  cter_ctf_file 


=== Typical usage ===

sxgui_cter GUI tool can be run as a stand-alone program or run from sxgui.py.

''' sxgui_cter.py  outdir_cter/partres.txt''' 


== Input ==
    cter_ctf_file:: CTF parameter file: This file is produced by sxcter and normally called partres.txt (default none)
